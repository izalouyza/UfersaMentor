# Backlog do Sistema

Este documento apresenta o backlog atualizado do **Sistema de Gerenciamento de Monitorias**, contendo os Casos de Uso mapeados durante a fase de engenharia de requisitos, sua respectiva priorização estrutural e o status atual de desenvolvimento do MVP (Release 1).

## Legenda de Status
* 🟢 **Implementado:** Funcionalidade concluída e integrada no MVP atual.
* 🟡 **Parcialmente Implementado:** Estrutura básica construída (backend ou frontend), mas aguardando refinamentos ou integração final.
* 🔴 **Não Implementado:** Funcionalidade mapeada para os próximos ciclos de desenvolvimento (Releases 2 e 3).
